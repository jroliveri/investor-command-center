# Roadmap

## MVP Complete

- Native C++ desktop app shell
- Dear ImGui UI
- SQLite local database
- Accounts CRUD
- Holdings CRUD
- Transactions CRUD
- Dividends CRUD
- Goals CRUD
- Watchlist CRUD
- Search/filter tables
- Dashboard summaries

## Next

- Basic reports using local records
- Settings page for database path and display preferences
- Better date entry controls
- CSV import/export for manual record entry
- More complete validation and duplicate detection

## Later

- Portfolio history snapshots
- Charts with ImPlot
- Backup and restore workflow
- Optional exportable reports

## Not Planned For This Private MVP

- Brokerage integrations
- Cloud sync
- Login
- Stock price APIs
- Buy/sell recommendations
- Financial advice
